# HTML_Game_1

A simple browser-based HTML game project.

## Project Goals

- Build a lightweight game using HTML, CSS, and JavaScript.
- Keep the project easy to run locally in a browser.
- Follow secure coding practices as the project grows.

## Planned Structure

```text
HTML_Game_1/
├── README.md
├── index.html
├── css/
│   └── styles.css
└── js/
    └── game.js
```

## Getting Started

Open `index.html` in a modern web browser once the game files are added.

## Security Notes

This project will follow the uploaded KRAFTON security guidance where applicable, including safe handling of user input, avoiding unsafe HTML injection, and keeping future dependencies minimal and reviewed.
